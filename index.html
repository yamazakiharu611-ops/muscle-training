About

Skills

  • HTML / CSS: 基本的なタグの使い方や、スマートフォンの画面サイズに対応したレイアウト（レスポンシブデザイン）を学習中です。
  • JavaScript: 簡単なイベント処理（ボタンをクリックした際の動きなど）を学んでいます。
  • ツール: VS Code / GitHub / GitHub Desktop

Works

Reflection

今回の演習を通じて、GitでのコミットやGitHubへのプッシュなど、Webサイトをインターネット上に公開するまでの一連のワークフローを体験することができました。

最初は専門用語や操作に戸惑うこともありましたが、実際に自分の作ったページがブラウザで表示されると達成感があります。今後はデザインだけでなく、JavaScriptを使った動きのある機能にもさらに挑戦してみたいです。
